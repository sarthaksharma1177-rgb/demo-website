## How to Open the Site

### Option 1: Open Locally
1. Clone or download this repository:
   ```bash
   git clone https://github.com/sarthaksharma1177-rgb/demo-website.git
   ```
2. Navigate to the folder:
   ```bash
   cd demo-website
   ```
3. Open `index.html` in your browser:
   - **Windows**: Double-click `index.html` or right-click → Open with → Chrome
   - **Mac/Linux**: Run `open index.html` (Mac) or `xdg-open index.html` (Linux)

### Option 2: Use VS Code Live Server
1. Open the project in VS Code
2. Install the **Live Server** extension
3. Right-click on `index.html` and select **"Open with Live Server"**
4. The site will open in your default browser at `http://localhost:5500`

### Option 3: Deploy Online
Since this is a private repo, GitHub Pages is not available. To host it online:
- Push to **Netlify** or **Vercel** (they support private repos)
- Or make this repo public to enable GitHub Pages
